# Technology Stack

## Core
- **Language:** Python (>= 3.11, < 3.13)
- **Scientific Computing:** NumPy, SciPy
- **Neuroimaging I/O:** Nibabel

## Domain Specific
- **Diffusion MRI:** DIPY
- **Visualization:** Fury
- **Tractogram Management:** Scilpy (internal modules)

## Infrastructure
- **CLI:** docopt
- **Packaging:** setuptools (build-backend), uv (installation recommendation)
- **Version Control:** Git
